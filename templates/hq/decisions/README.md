# Decisions

Immutable decision records. One file per non-trivial choice: `YYYY-MM-DD-<slug>.md`.

**Format:**
```markdown
# YYYY-MM-DD — <Decision title>

## Context
<What prompted this decision>

## Options considered
- Option A: ...
- Option B: ...

## Decision
<What was decided and why>

## Consequences
<What this means going forward>

## Supersedes
<Link to prior decision if this replaces one>
```

Never edit past decisions. Supersession via new files only.
