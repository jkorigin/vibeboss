## Approach

Single bash script + templates directory. No external dependencies beyond bash and python3 (both present on any supported platform that can run Claude Code). Placeholder substitution via bash `${var//pattern/replacement}` â€” no sed/awk required, no regex metacharacter risk since bash `//` is literal glob matching.

### Step 1: Template tree
Create `vibeboss/templates/hq/` and `vibeboss/templates/labs/` mirroring the runtime workspace, with `{{PLACEHOLDER}}` values where user-specific content would appear.

Key decisions:
- HQ CLAUDE.md: generic version referencing `{{LEAD_NAME}}`, `{{WORKSPACE}}`, etc.
