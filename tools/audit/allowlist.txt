# One regex per line. The sensitivity audit (tools/audit/audit.sh) will skip any
# finding whose snippet matches one of these patterns.
#
# Rules:
#   - Lines starting with # are comments.
#   - Blank lines are ignored.
#   - Each line is an ERE regex. Use \\ to escape backslashes, \. for literal dot.
#   - Patterns match anywhere in the offending line (not anchored).
#   - DO NOT add the literal sensitive data here — that's the same circular-leak
#     trap that motivated this whole script. Allowlist *patterns* and known-public
#     references only.
